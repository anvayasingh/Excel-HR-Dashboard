# HR Analytics & Attrition Dashboard (Excel Project)

This project presents an interactive Excel dashboard that analyzes key aspects of employee data to understand attrition patterns, workforce distribution, and performance indicators. It offers valuable insights into the organizationâ€™s human resource dynamics through visual analytics and KPI tracking.

# Key Features & Insights

Attrition Analysis: Breakdown of employee attrition by age group, department, job role, and education level.

Demographic Insights: Visual representation of gender and marital status distributions within the workforce.

Performance Evaluation: Integration of KPI and rating data to assess employee satisfaction and performance trends.

Department-Wise Trends: Department-specific attrition and performance visualization for targeted HR planning.

Interactive Dashboard: Centralized DASHBOARD sheet summarizing all HR metrics through charts, slicers, and KPIs for straightforward interpretation.

# Tech Stack

Tool: Microsoft Excel

Techniques: Pivot tables, slicers, conditional formatting, dynamic charts, and data visualization

# Objective

To identify patterns in employee attrition, highlight workforce demographics, and support HR teams in developing data-driven retention and engagement strategies.
